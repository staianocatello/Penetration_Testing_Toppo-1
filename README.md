# Penetration_Testing_Toppo-1

## Panoramica

Questa repository raccoglie l'intero lavoro di penetration testing eseguito sulla macchina vulnerabile denominata **Toppo: 1**.  
L'obiettivo è individuare e sfruttare le debolezze presenti nel sistema per evidenziare l'importanza di misure di sicurezza informatica efficaci.

---

## Contenuti della repository

- Documentazione completa del processo di penetration testing (contenuta nella repository)  
- Risultati dettagliati e analisi delle vulnerabilità riscontrate  
- Materiale di supporto per la presentazione
